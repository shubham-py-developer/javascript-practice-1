📘 JavaScript Practice Project

This repository contains my structured JavaScript practice exercises along with a basic backend setup for serving static and dynamic content.

The goal of this project is to practice core JavaScript concepts, improve problem-solving skills, and maintain clean project organization using Git and VS Code.

How JavaScript Practice Files Are Organized

Each file inside the js/ folder represents a learning module.

Each file may contain multiple problems.

Since only one problem is executed at a time, previous problems are commented out while testing new ones.

This structure helps maintain all related practice in one place without creating excessive files.


How to Run the Project:
Run Python Static Server via terminal
python3 server_static.py
This serves static files like HTML, CSS, and JS.
now on browser right click -inspect-console and now refresh page and click on file button which you want to run on console!


if you want to run file on your terminal simply:
node file_name.js
